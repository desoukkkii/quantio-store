# ShopMate - Simple E-Commerce Website

A beginner-friendly online store built with HTML, CSS, and JavaScript. No frameworks - just pure code!

## Features

- Product listing with images, prices, and categories
- Search products by name
- Filter products by category
- Add/remove items from cart
- Increase/decrease product quantity
- Auto-calculates cart total
- Saves cart data in your browser
- Works on mobile, tablet, and desktop

## How to Use

1. Download all files (index.html, style.css, script.js)
2. Put them in the same folder
3. Double-click index.html to open in your browser

That's it! No installation needed.

## Project Structure

shopmate/
├── index.html # Main page structure
├── style.css # All styling
└── script.js # All functionality


## How It Works

- Products load from [DummyJSON API](https://dummyjson.com/products)
- Your cart saves automatically in your browser
- Search and filter work in real-time
- Click the shopping bag icon to view your cart

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript ES6+
- Font Awesome Icons

## Customize It

**Change colors:** Edit the `:root` colors in style.css

**Change API limit:** Edit line 10 in script.js

## Live Demo

Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge)
